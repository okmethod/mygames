# mygames

## やること
pygameを使用して、pythonでゲームを実装する。

## 目的
- オブジェクト指向プログラミングの練習をする。
- MVCモデルなど、設計や再利用の練習をする。
- 実装を通して、ゲームへの理解を深める。
